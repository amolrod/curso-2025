Información sobre Accesibilidad

Este formulario incluye varias medidas de accesibilidad:

  • Etiquetas claras: Cada campo tiene su etiqueta asociada con el atributo for, para que los lectores de pantalla sepan qué es cada cosa.
  • Ayudas contextuales: Los textos pequeños debajo de cada campo dan información extra sobre qué poner.
  • Validación accesible: Los errores se anuncian con role="alert" y aria-live="polite" para que los lectores de pantalla los detecten.
  • Navegación por teclado: Puedes rellenar todo el formulario solo con el teclado (Tab, Shift+Tab, Espacio, Enter).
  • Foco visible: Cuando navegas con teclado, se ve claramente dónde estás.
  • Campos obligatorios marcados: Los asteriscos tienen un aria-label para que se anuncie "campo obligatorio".
