PELÍCULAS DE TERROR CLÁSICAS
Proyecto web accesible sobre películas de terror con temática Halloween

QUÉ ES
Página web de catálogo de películas de terror clásicas que demuestra principios de accesibilidad web siguiendo las pautas WCAG 2.1.​

PÁGINAS DEL SITIO

index.html - inicio con película destacada
catalogo.html - listado de películas por década
pelicula.html - ficha detallada de Nosferatu (1922)
contacto.html - formulario de contacto

ACCESIBILIDAD IMPLEMENTADA

Perceptible
Texto alternativo en todas las imágenes
Contraste de colores adecuado (mínimo 4.5:1)
Encabezados en orden jerárquico (h1 → h2 → h3 → h4)
Descripciones textuales del contenido visual
Operable
Navegación completa por teclado (Tab + Enter)
Enlace "Saltar al contenido" visible al hacer Tab
Foco visible en todos los elementos interactivos
Botones y enlaces de mínimo 44x44px
Comprensible
Lenguaje claro y directo
Etiquetas asociadas a todos los campos del formulario
Textos de ayuda contextual
Indicador de página actual en menú
Robusto
HTML5 semántico válido
Roles ARIA: banner, navigation, main, contentinfo
Formularios accesibles con labels y fieldset

USUARIOS CONTEMPLADOS

Discapacidad visual: texto alternativo, alto contraste, estructura para lectores de pantalla
Discapacidad auditiva: descripciones textuales del contenido audiovisual
Discapacidad motora: navegación por teclado, botones grandes
Discapacidad cognitiva: estructura simple, lenguaje directo, navegación consistente

TECNOLOGÍAS

HTML5 semántico + CSS3 + JavaScript 

INSTALACIÓN

No requiere instalación. Abrir index.html en cualquier navegador.
